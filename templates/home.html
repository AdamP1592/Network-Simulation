This is my Neuron Culture Simulation!
It was made using hodgkin huxley neuron models connected using tsodyks markram synapse models. I am actively searching for a job.


This simulation was built from scratch! For the server, I used flask and the entire model was made to be stored in json or a nosql database. The entire simulation is written in Python, only incorporating Shapely and Numpy for synapse generation!


The simulation is in the simulation tab. You can click the main graph to add electrodes, click neurons to see properties, or hover over neurons and synapses to see their basic information.

To set electrode currents, right click on the electrodes and hit enter when you've filled out each of the fields
