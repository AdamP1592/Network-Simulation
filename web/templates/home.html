This is my Neuron Culture Simulation!
It was made using Hodgkin-Huxley neuron models connected via Tsodyks-Markram synapse models, with connections formed through probabilistic synaptogenesis.
I am actively searching for a job.


This simulation was built from scratch! The backend uses Flask, and the model is designed to be stored in JSON or a NoSQL database. The simulation is written in Python, using Shapely and NumPy for synapse generation.


You can click the main graph to add electrodes, click neurons to see properties, or hover over neurons and synapses to see their basic information.
To set electrode currents, right-click on the electrodes and hit enter when you've filled out each of the fields.
For performance reasons, network complexity is limited and synaptic strengths are set high to ensure visible neural activity. These settings highlight interactive features, not biological accuracy.
